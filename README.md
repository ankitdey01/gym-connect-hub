# Gym Connect Hub 💪

A modern gym/community web app built with **React + TypeScript + Vite**, styled with **Tailwind CSS** and **shadcn/ui (Radix UI)** components.

> Repo: `ankitdey01/gym-connect-hub`

---

## Tech Stack 🧑‍💻

- **Frontend:** React 18, TypeScript
- **Build tooling:** Vite
- **Routing:** React Router DOM
- **UI:** shadcn/ui + Radix UI, Lucide icons
- **Styling:** Tailwind CSS (+ tailwindcss-animate, typography)
- **Forms & validation:** React Hook Form, Zod, @hookform/resolvers
- **Data fetching/state:** @tanstack/react-query
- **Charts:** Recharts
- **Animation:** Framer Motion
- **Testing:** Vitest, Testing Library

## Getting Started 📃

### Prerequisites
- **Node.js** (recommended: latest LTS)
- **npm** (or your preferred package manager)

### Install
```bash
npm install
```

### Run locally
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Build (development mode)
```bash
npm run build:dev
```

### Preview production build
```bash
npm run preview
```

---

## Scripts

- `dev` – start the dev server
- `build` – create a production build
- `build:dev` – create a development-mode build
- `lint` – run ESLint
- `test` – run tests once (Vitest)
- `test:watch` – run tests in watch mode

If the app uses environment variables, place them in a `.env` file at the project root.

Vite exposes variables prefixed with `VITE_`, for example:
